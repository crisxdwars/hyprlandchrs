# **Resources To Install**

    sudo pacman -S waybar gpu-screen-recorder wl-clipboard slurp ttf-fira-code ttf-fira-code noto-fonts-cjk zip unzip pavucontrol

### ****Resource Guide****
   waybar = Its The Top Bar
   gpu-screen-recorder = Graphics Based Screen Recorder
   wl-clipboard & slurp = Resources For Screenshot System
   ttf-hack-nerd = Font Used For Waybar
   ttf-fira-code = Font Used For Kitty (Terminal)
   noto-fonts-cjk = Font Support For Chinese, Japanese, Korean
   zip = Used to Zip & Unzip Files
   pavucontrol = Output(Speakers) & Input(Microphone) Controls
