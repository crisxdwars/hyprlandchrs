# **Resources Instalations**
    ## Install Everything at once
    ```sudo pacman -S waybar gpu-screen-recorder wl-clipboard slurp ttf-hack-nerd zip unzip pavucontrol```
    
    ## Install each

    ***Top Bar***
        ```bash
        sudo pacman -S waybar```
    ***Win + ALT + R Screen Recorder***
        ```bash 
        sudo pacman -S gpu-screen-recorder```
    ***Top Bar***
    ```sudo pacman -S wl-clipboard slurp```
    ***Top Bar***
    ```sudo pacman -S ttf-hack-nerd```
    ***Top Bar***
    ```sudo pacman -S zip unzip```
    ***Top Bar***
    ```sudo pacman -S pavucontrol```
